Python3 的六个标准数据类型中：

不可变数据（3 个）：Number（数字）、String（字符串）、Tuple（元组）；  
可变数据（3 个）：List（列表）、Dictionary（字典）、Set（集合）。

## 一、Number（数字）
### 1.int
在Python 3里，只有一种整数类型 int，表示为长整型，没有 python2 中的 Long。
#### 进制
* 十进制
* 二进制：0b或0B
* 八进制：0o或0O
* 十六进制：0x或0X
#### 各进制之间的转换
* bin(x):将x转换成二进制，返回值是str
* oct(x):将x转换成八进制，返回值是str
* int(x):将x转换成十进制，返回值是int,可以将字符串类型的整数转换成int类型
* hex(x):将x转换成十六进制,返回值是str

### 2.float
数学中的小数  
科学计数：1.2e8 1.2e-8  
浮点的不确定性：浮点数在计算机中只能做到近似存储  

### 3.bool
True 1  
False 0  
0 ,0.0,空 等会转换成False，其他值均转换成True  

### 4.complex
虚数用j表示，c=4-5j  
c.real：显示实部  
c.imag：显示虚部

## 三、String（字符串）
创建：单引号，双引号  
```
str1 = 'hello world'
str2 = "hello world"
```

索引：访问序列中的某一个元素  
格式：字符串名[index]  
范围：-len（）--------len（）-1   
index:  
正数：方向从左到右进行索引  
负数：方向从右到左进行索引  

切片：获取序列中 一定区域的元素  
**str[0:2]，表示的是截取前两元素，左闭右开**  
语法 ：字符串[start:end:step]  
start: 起始位置：正数 左到右，否则从右到左，默认0  
end: 终止位置：不包含end，默认len(a)  
step：代表获取元素的方向，如果是正数，代表从左到右切取，要求start位置必须在end位置的左侧  
如果是负数，代表从右到左切取，要求start位置必须在end位置的右侧

## 四、List（列表）
列表 list -- 容器类型，能够存储多个元素的类型  
添加  
(1) li.append（x）：向列表尾部添加一个元素  
(2) li.insert(index，object)：向列表的指定位置插入一个元素  
```
1)index：要插入的位置

2)object：要插入的元素
```
(3) li.extend（li2）：合并，将参数列表中的元素合并到原列表中  

列表的相关方法---删除  
(1) li.pop（index）：删除指定位置的一个元素  
```
1)index：要删除的位置，index省略时，删除最后一个元素

2)返回值是刚刚删除元素的内容
```
(2) li.remove（object）：删除指定内容的一个元素，object是要删除的元素，只能删除第一个  
(3) li.clear()：清空列表中的所有元素  
直接赋值：其实就是对象的引用（别名）。  
浅拷贝(copy)：拷贝父对象，不会拷贝对象的内部的子对象。  
深拷贝(deepcopy)： copy 模块的 deepcopy 方法，完全拷贝了父对象及其子对象。  

## 五、Tuple（元组）
本质上：只读的list  
t5 = ('1',)  # 定义一个元素时，需要在元素后加上逗号  
t6 = (1,'asc',False,[1,2,3],('a','b'))  # 元组中可以包含多种数据类型的元素  
t7 = (1,'asc',False,[1,2,3],('a','b'),1,1,1)  # 元组中的元素是可以重复的，并且是有顺序的  
元组是不可变数据类型，元组中的元素不可以进行修改的,如果元组中的元素是可变类型，可变类型中的元素可以进行修改  

## 六、Set（集合）
集合 set  
集合的特点：  
确定性:集合中的元素是不可变数据类型  
互异性：集合中的元素是不能重复的  
无序性：集合是无序的，不能使用索引，只能遍历  

定义空的集合，只能使用set()  

集合是可变数据类型，集合中的元素是不可变数据类型  

add向集合中添加指定的元素，如果重，不添加  

## 七、Dictionary（字典）
映射类型数据：根据“键” 映射到“值”  
一个元素分为2部分 key:value 键值对  

字典通过键值对key：value的方式存储数据  

字典中key不可以重复,value可以重复  

字典中的key必须是不可变数据类型,value不做限制
